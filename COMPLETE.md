```

```

```css
In styles.css, update:
--primary: #6C3BFF;        /* Your color */
--primary-light: #A366FF;  /* Lighter */
--primary-dark: #5530CC;   /* Darker */
```

## 💡 Pro Tips

1. **Test Locally First**: Open in browser before deploying
2. **Mobile Testing**: Use actual phone, not just browser resize
3. **Color Coordination**: Keep at least 3 colors consistent
4. **Image Optimization**: Compress before uploading
5. **Browser Cache**: Clear when styles don't update
6. **Backup Files**: Keep original copy before editing
7. **Use Inspector**: F12 to debug CSS on the fly
8. **Performance**: No dependencies = blazing fast
9. **Accessibility**: Test with keyboard navigation
10. **Mobile-First**: Design for phone first, then expand

## 🔧 Troubleshooting

### Images not showing?
→ Check images/ folder path
→ Verify filename is correct
→ Use full paths if needed

### Styles look wrong?
→ Clear browser cache (Ctrl+Shift+Delete)
→ Reload with Ctrl+Shift+R
→ Check file path in HTML

### Animations stuttering?
→ Disable browser extensions
→ Check system resources
→ Test on different browser

### Forms not working?
→ They're frontend only (no backend)
→ Console shows all errors (F12)
→ Need API connection for data saving
